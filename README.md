# NPM library boilerplate

- AVA
- Babel(using env)
- Flowtype
- ESLint extends StandardJS (and prettier-eslint)
- 

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From repo

```bash
sao magitek-telescope/npm-lib-boilerplate lib-name
```
